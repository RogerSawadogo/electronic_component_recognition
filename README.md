# 🔌 Real-Time Electronic Component Detection with YOLO

A computer vision system that detects and classifies electronic components (mcu, sensors, ICs) on a breadboard in real-time using a webcam.

## 🎯 Performance

The custom-trained YOLOv8 model achieved a mean Average Precision (mAP@50) of **~92%** on the validation set.

## 🛠️ Tech Stack

- **Framework:** Ultralytics YOLO
- **Language:** Python
- **Libraries:** OpenCV, NumPy
- **Data:** Custom-built dataset of annotated electronic components.
